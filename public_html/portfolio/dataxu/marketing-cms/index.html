How I Built a Custom CMS for a Marketing Team

Role: Prototype, Interaction Design, User Testing, Development

Summary

I designed and developed a custom CMS (Content Management System) for the dataxu marketing team. This modular template system allowed marketers to easily create new landing pages using pre-built layouts.

Challenge

The dataxu marketing team was using an outdated template for content marketing landing pages. The process to create new pages was difficult, slow, and manual. The resulting pages were not optimized for all devices, and the layouts were rigid and inflexible.

Solution

I created a new CMS for marketers to quickly and easily create beautiful landing pages. The system is based on a single modular template with bootstrap at its core. Content marketers can easily create new pages without wasting any time messing with the layout.

The system was designed to easily allow for A/B testing to increase conversions. In this scenario, option 1 offers a free PDF, with an optional contact form below. While option 2 requires a user to fill out the form to get content.

The user interface for this popular starter template was designed for content promotion and features a simple three-column layout.

The CMS has custom variables that can be adjusted for each page. This allows marketers to easily hide or show entire sections with the flip of a switch.

The first round of pages created with the new template were each customized with unique emoji in the header and custom CTA buttons. The emoji were a "humanizing" part of the brand that was intended to resonate with a younger audience.

The final page is fully responsive and enables marketers to produce unique pages without digging into any code or advanced page layout. This also significantly reduced the time needed to design and develop future pages, allowing me to focus more on strategy and less on production.

Results

This enabled marketers at dataxu to easily build beautiful landing pages, and ultimately increase leads and revenue for the business.

The team was able to set up a handful of A/B tests to and saw the following improvements:

  • 60% increase in conversions
  • 30% increase in average page speed
